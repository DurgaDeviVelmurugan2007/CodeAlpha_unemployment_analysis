# CodeAlpha_unemployment_analysis

Unemployment Analysis in India

Project Overview

This project analyzes unemployment trends in India using data analysis and visualization techniques. The objective is to identify patterns in unemployment rates across different states, compare urban and rural unemployment, and study the impact of COVID-19 on employment.

Objectives

- Perform data cleaning and preprocessing.
- Analyze unemployment rates across Indian states.
- Compare unemployment trends in urban and rural areas.
- Study the impact of COVID-19 on unemployment.
- Visualize unemployment patterns using charts and graphs.
- Generate insights to support decision-making.

Dataset

Dataset: Unemployment in India

Dataset Features

- Region
- Date
- Frequency
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)
- Area (Urban/Rural)

Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

Data Cleaning Process

The following preprocessing steps were performed:

1. Removed missing values.
2. Removed extra spaces from column names.
3. Cleaned text fields.
4. Converted the Date column into datetime format.
5. Removed invalid records after conversion.

Exploratory Data Analysis

The analysis includes:

- Statistical summary of the dataset.
- State-wise unemployment analysis.
- Monthly unemployment trend analysis.
- Urban vs Rural unemployment comparison.
- COVID-19 impact analysis.
- Correlation analysis.

Visualizations

The project generates:

- Top 10 States with Highest Unemployment Rate (Bar Chart)
- Monthly Unemployment Trend (Line Chart)
- Urban vs Rural Comparison (Box Plot)
- COVID-19 Impact Analysis (Line Plot)
- State-wise Unemployment Heatmap
- Correlation Matrix

Key Findings

- Unemployment rates vary significantly across states.
- Certain states experience consistently higher unemployment rates.
- COVID-19 caused a noticeable increase in unemployment during 2020.
- Urban and rural regions show different unemployment patterns.
- Labour participation rate influences unemployment trends.

Conclusion

The analysis demonstrates the impact of economic and social factors on unemployment in India. The COVID-19 pandemic significantly affected employment levels across the country. Data visualization and trend analysis help in understanding these changes and can support future policy planning.
