# CodeAlpha_unemployment_analysis

📊 Unemployment Analysis in India

A Data Analysis project that analyzes unemployment trends across India, studies the impact of COVID-19, and visualizes unemployment patterns using Python, Pandas, Matplotlib, and Seaborn.

---

📌 Features

- Load and explore the unemployment dataset
- Perform data cleaning and preprocessing
- Analyze unemployment rates across different states
- Compare Urban and Rural unemployment trends
- Study the impact of COVID-19 on unemployment
- Generate statistical summaries and insights
- Visualize results using:
  - Top 10 States with Highest Unemployment Rate
  - Monthly Unemployment Trend
  - Urban vs Rural Comparison
  - COVID-19 Impact Analysis
  - State-wise Unemployment Heatmap
  - Correlation Matrix
- Generate meaningful conclusions from the analysis
